## 修饰符

### 概述  

> 修饰符用来定义类、方法或者变量，通常放在语句的最前端。

### 划分点

* 访问修饰符
* 非访问修饰符 

> * ####  访问修饰符
>
> > default （缺省）：缺省访问修饰符声明的变量和方法，对同一包内的类是可见的。
> >
> > public：在同一类内可见。
> >
> > private：对所有类可见。
> >
> > protected： 对同一包内的类和所有子类可见。
>
